‚ÄãHeuristic Evaluation Summary
‚ÄãThe evaluation rates application compliance across several core usability heuristics:  
‚Äã1. Visibility of System Status
Instagram (GOOD/STRONG): Provides clear feedback with loading indicators, interactive stories progress indicators, and instantaneous real-time feed updates.  
‚ÄãAmazon (GOOD/STRONG): Delivers highly clear search results, prominent delivery information layout, clear stock status alerts, and exact progress tracking dashboards. 
 
‚Äã2. Match Between System and the Real World
Instagram (GOOD/STRONG): Uses familiar language, standard social networking icons, and highly relatable real-world metaphors (e.g., physical camera, heart for liking, and paper plane for direct messaging).  
‚ÄãAmazon (GOOD/STRONG): Employs widely recognized transactional terms ("Add to Cart", "Buy Now", "Delivery"), universally standardized e-commerce icons, and a layout that mimics a real-world shopping flow. 
 
‚Äã3. User Control and Freedom
‚ÄãInstagram (NEEDS IMPROVEMENT): Offers very limited undo options or recovery paths for critical user actions, such as accidentally deleting posts or wiping out direct messages.  
‚ÄãAmazon (GOOD/STRONG): Features seamless post-purchase workflows including easy cancellations (return orders), quick cart edits, easy item removal, and the ability to modify delivery options dynamically. 
 
‚Äã4. Consistency and Standards
‚ÄãInstagram (GOOD/STRONG): Strictly follows standard mobile UI design guidelines and social media industry standards for layout, navigation, and gestures/interactions.  
‚ÄãAmazon (GOOD/STRONG): Adheres comprehensively to global e-commerce patterns and web standards that users have already built familiarity with across multiple platforms.
  
‚Äã5. Error Prevention
‚ÄãInstagram (NEEDS IMPROVEMENT): Accidental double-tap likes or sending messages/requests to the wrong DMs can easily happen due to a lack of friction or confirmation steps.  
‚ÄãAmazon (GOOD/STRONG): Explicitly confirms critical details before placing final orders, displays prominent interactive warnings for limited stock availability, and flags user eligibility issues actively. 
 
‚Äã6. Recognition Rather Than Recall
‚ÄãInstagram (GOOD/STRONG): Utilizes highly persistent bottom icons, explicit tab layouts, and contextual text suggestions to minimize cognitive memory load and keep system features discoverable.  
‚ÄãAmazon (GOOD/STRONG): Provides robust live search suggestions, recent item search history lists, delivery addresses, and saved watchlists that remain easily accessible at all touchpoints.
